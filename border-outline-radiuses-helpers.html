Border helpers

Applies border: 0 to element. 
<span class="no-outline">Lorem ipsum dolor</span>
Applies outline: 0 to element. Please use it really if you need to, removing outline is 99% a bad idea.

Border radiuses

<div class="rounded">
Applies standard border-radius to element (value defined with $global-border-radius).
If you need to cancel border-radius, you might use `rounded0`, `rounded0-left` and `rounded0-right` helpers. 
<span class="rounded50">
Applies border-radius: 50% to element.
