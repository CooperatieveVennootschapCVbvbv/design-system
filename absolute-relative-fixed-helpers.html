<span class="relative">Lorem ipsum dolor</span>
Applies position: relative to element. 
<span class="fixed">Lorem ipsum dolor</span>
Applies position: fixed to element. 
<span class="absolute">Lorem ipsum dolor</span>
Applies position: absolute to element + z-index: 1. 
<span class="centered-absolute">Lorem ipsum dolor</span>
Centers absolute-positionned element via top: 50%; left: 50%; transform: translate(-50%, -50%);. 
<span class="centered-absolute-horizontal">Lorem ipsum dolor</span>
Centers absolute-positionned element via left: 50%; transform: translate(-50%, 0);. 
<span class="covered-absolute">Lorem ipsum dolor</span>
covered-absolute class applies position: absolute; top: 0; right: 0; bottom: 0; left: 0;. 
<span class="top-right">Lorem ipsum dolor</span>
Applies top: 0; right: 0; to absolute-positionned element. 
<span class="bottom">Lorem ipsum dolor</span>
Applies bottom: 0; to absolute-positionned element.

In case of, there are helpers on mobile like onmobile-static which can cancel fixed/absolute positionning.
